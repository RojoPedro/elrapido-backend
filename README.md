# Pizza CMS Backend 🍕
Sistema di gestione per pizzerie realizzato con Laravel 11.

## Funzionalità
- **Autenticazione:** Gestita con Laravel Sanctum.
- **CRUD Pizze:** Gestione completa del menu con transazioni DB.
- **CRUD Ingredienti:** Sistema di protezione (impedisce l'eliminazione se l'ingrediente è in uso).
- **Security:** Implementate Policy di autorizzazione.
